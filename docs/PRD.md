# MoodCast MVP — Product Requirements (Demo)

## Vision
AI-powered emotional social network. Users share how they FEEL, get matched into 24-hour rooms, earn MoodCoins from reactions.

## Demo personas
- Stressed student (Rahul)
- Anxious professional (Priya)
- Demo investor walkthrough user

## User stories (MVP)

1. As a user, I can register with email so I have a persistent account.
2. As a user, I can check in daily with free text so I get mood-matched.
3. As a user, I am placed in a room of up to 12 similar moods so I feel understood.
4. As a user, I can post thoughts and react (not comment) so the space stays safe.
5. As a user, I earn MoodCoins when others react to my posts.
6. As a user, I can view my wallet and demo-redeem coins.

## Acceptance criteria

- [ ] Register/login works
- [ ] Check-in creates mood label and assigns room
- [ ] Room shows members (anonymous), posts, expiry countdown
- [ ] Reactions update counts and credit coins to author
- [ ] Room inaccessible after 24h
- [ ] Wallet shows balance and transaction history
